# Бот итогов дня — Фаза 1

Собирает ответы сотрудника на 5 вопросов и формирует структурированный итог дня.

## Что делает (Фаза 1)
- Команда `/report` запускает опрос
- Бот задаёт 5 вопросов по очереди
- Собирает ответы и выдаёт аккуратный структурированный итог
- Без AI и без WEEEK (это следующие фазы)

## Вопросы
1. Над какими проектами сегодня работал?
2. Что конкретно делал и что сделал по каждому проекту?
3. Были трудности или нерешённые проблемы?
4. Узнал или попробовал что-то новое сегодня?
5. Координация, встречи, прочее?

## Команды бота
- `/start` — приветствие и справка
- `/report` — начать сбор итога дня
- `/cancel` — прервать текущий опрос

## Локальный запуск (для теста)
```
pip install -r requirements.txt
set BOT_TOKEN=твой_токен_от_BotFather   (Windows)
export BOT_TOKEN=твой_токен_от_BotFather  (Linux/Mac)
python bot.py
```

## Деплой на Render

1. Создай бота через @BotFather в Telegram, получи токен
2. Залей этот код в репозиторий GitHub
3. На Render: New → Web Service → подключи репозиторий
4. Настройки:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python bot.py`
5. Environment → добавь переменную:
   - `BOT_TOKEN` = твой токен от BotFather
6. Render сам задаёт переменную `PORT` — keep-alive сервер её подхватит

## Keep-alive (чтобы Render не усыплял бесплатный сервис)
1. Скопируй URL сервиса с Render (вида https://имя.onrender.com)
2. На UptimeRobot создай HTTP(s) монитор на этот URL с интервалом 5 минут
3. Это держит сервис активным
